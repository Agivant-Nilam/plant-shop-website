# Plant Shop Website

A modern, responsive plant shop website built with HTML and CSS. Features a clean design with a hero section showcasing a monstera plant.

## Features

- Clean and modern design
- Fully responsive layout
- Interactive navigation
- Beautiful plant imagery
- Shopping functionality

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Poppins)

## Setup

1. Clone the repository
2. Open `plant_shop.html` in a web browser

## Preview

The website features a beautiful hero section with:
- Modern typography
- Responsive design
- Interactive elements
- Plant imagery
